Controls:
WASD - Move Player
Q - Move Player Up
Z - Move Player Down
Shift - Move faster
Esc - Quit

B - Draw arrow w/ Depth Test Off
N - Draw arrow w/ Depth Test On
M - Draw arrow w/ Xray on

~ - Show Console

O - Show Origin Axes
P - Use time based shader on Sphere

How each feature is demonstrated:

Cube and sphere are in the world next to each other. 

Edit the fvf and fvf2 shaders under Data/Shaders to create compile errors.
